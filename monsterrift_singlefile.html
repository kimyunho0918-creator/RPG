게임 설명
스타터 몬스터 하나를 고르고 웨이브를 돌파해. 전투 후에는 보상 중 하나를 선택하고, 보스 웨이브를 넘길수록 더 강해진다.
왜 한 파일 버전인가
GitHub 업로드가 쉬워. 파일 하나만 올려도 바로 실행되니까 관리가 단순하고 공유도 쉽다.
중요한 점
요청한 “포켓몬 그대로 사용”은 저작권과 IP 문제 때문에 넣지 않았어. 대신 배포 가능한 오리지널 캐릭터와 분위기를 넣어뒀다.

게임 소개

Monster Rift는 포켓몬식 수집 전투의 감각 대신, 법적으로 안전하게 쓸 수 있는 오리지널 몬스터로 만든 로그라이크 RPG야.

기본 흐름은 전투 → 승리 → 보상 선택 → 다음 웨이브 구조고, 일정 웨이브마다 보스가 등장해. 공격, 가드, 스킬, 회복약을 적절히 섞는 게 핵심이야.
